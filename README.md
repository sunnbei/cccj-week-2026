## 5. 参加費・招待設計（会議での議論）

- 会員 **5,000円**（例会並み、来てほしいのは会員）／非会員・外部 10,000円（ルール上決めるが実質招待者オンリー）
- **一般販売しない**（好ましくない参加者を断れなくなるため）。紹介・招待枠のみ → 招待が価値になる
- 協賛招待枠だけで100〜160名が埋まる試算 → **会員枠を別途確保**する必要
- 経済団体・JC等へは礼儀として案内枠（10〜15名程度）を会の枠から確保

## 8. 資料(6/11)と会議(6/15)の差分 — 要調整事項

会議で出た方向性が、現行ドラフトにまだ反映されていない。発信前に確定・修正が必要。

| 論点 | 資料(6/11)の現状 | 会議(6/15)の方向 |
|---|---|---|
| 参加費 | 20,000円／会員15,000円 | 会員5,000円・招待者オンリー（一般販売なし） |
| 協賛階層 | 3階層（ダイヤ/ゴールド/シルバー）計50社 | **社数を増やす**（80〜100社、会員の95%）。**ブロンズ（10万）追加**検討 |
| 協賛金額 | 100/50/25万 | 例: 50万×20・30万×30・10万×50＝約2,400万 など下げて数を増やす案 |
| 基調講演 | 13:10〜14:10（約1時間） | 20〜30分で十分 |
| パネルディスカッション | （分科会内で想定） | 基調講演後のパネルは今回なし |
| ハンドブック | 事後報告書イメージ | **事前配布**に変更（協賛メリット重視） |
| VIP席 | 晩宴会席にVIP/前列 | 今回VIP席なし → 該当箇所カット |

> ブロンズはPPTXのハンドブック掲載欄に既出（ダイヤ/ゴールド/シルバー/**ブロンズ**）。階層追加は整合可能。
